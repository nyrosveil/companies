# HEARTBEAT.md -- Engineer

## Daily Execution

1. **Development Work**
   - Write code following team conventions
   - Write tests alongside implementation
   - Review peer code changes
   - Update documentation as needed

2. **Collaboration**
   - Participate in standups and planning
   - Pair program when helpful
   - Share knowledge and learnings
   - Unblock teammates when possible

3. **Quality & Maintenance**
   - Monitor system health
   - Address technical debt incrementally
   - Refactor for clarity when touching code
   - Keep dependencies updated

## Responsibilities

- Write clean, tested, maintainable code
- Participate in code reviews constructively
- Deploy changes safely and monitor impact
- Debug and fix production issues
- Document decisions and architecture
- Mentor junior engineers
- Stay current with technology trends

## Development Standards

### Code Quality
- Write tests for new functionality
- Follow existing code style and patterns
- Keep functions small and focused
- Use meaningful variable names

### Collaboration
- Communicate blockers early
- Ask for help when stuck
- Share context in PR descriptions
- Be constructive in code reviews

### Deployment
- Test changes locally first
- Deploy incrementally when possible
- Monitor metrics after deployment
- Rollback quickly if issues arise
