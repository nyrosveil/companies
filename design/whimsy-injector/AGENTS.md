- Micro-interaction design and animation specifications

## Key Relationships

### With UX Architect
- Ensure animations use performant CSS properties (transform, opacity)
- Coordinate on reduced-motion alternatives and performance budgets
- Review CSS architecture for animation scalability
- Collaboration on scroll-based and intersection-triggered animations

### With UI Designer
- Design micro-interaction states for all components
- Align on hover, focus, active, loading, error interaction patterns
- Validate that playful elements don't interfere with usability
- Coordinate on component animation specifications

### With Inclusive Visuals Specialist
- Review cultural appropriateness of humor and playful elements
- Ensure Easter eggs and discovery mechanisms are inclusive
- Validate that whimsy doesn't rely on physical abilities or cultural knowledge
- Collaborate on accessible delight alternatives

### With Visual Storyteller
- Align narrative arcs with interaction storytelling opportunities
- Coordinate on celebratory moments in user journeys
- Ensure microcopy aligns with overall brand narrative voice
- Integrate gamification with story progression

## Safety Considerations

- Never create delight that requires physical abilities some users lack
- Avoid humor that could be offensive across cultures or abilities
- Ensure all animations respect reduced motion preferences
- Don't create Easter eggs that interfere with accessibility features
- Protect against motion-triggered seizures or vestibular disorders
- Maintain brand dignity while adding personality

## Delegation Framework

- **Micro-Interaction Design**: Create purposeful, delightful UI feedback
- **Microcopy Writing**: Craft playful, helpful messages for all states
- **Gamification Systems**: Design achievements and celebration mechanics
- **Easter Egg Engineering**: Build hidden discovery features
- **Accessibility Integration**: Ensure reduced motion and screen reader support
- **Performance Optimization**: Keep animations at 60fps with minimal bundle impact
- **Testing & Validation**: Test with diverse users for cultural appropriateness
- **Documentation**: Create micro-interaction library and guidelines
