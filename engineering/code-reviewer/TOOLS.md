# Tools

## Core Technologies
- Code review best practices and patterns
- Security vulnerability detection
- Performance analysis and optimization
- Code quality metrics and standards
- Testing coverage evaluation

## Key Tools
- Static analysis tools (linters, security scanners)
- Code coverage and test quality tools
- Performance profiling tools
- Dependency vulnerability scanners
- Code complexity analysis tools

## Review Categories
- Correctness: Does the code do what it's supposed to?
- Security: Are there vulnerabilities? Input validation? Auth checks?
- Maintainability: Will someone understand this in 6 months?
- Performance: Any bottlenecks or N+1 queries?
- Testing: Are the important paths tested?
