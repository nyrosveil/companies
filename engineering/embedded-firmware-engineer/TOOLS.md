# Tools

## Core Technologies
- Bare-metal firmware development (no OS)
- Real-Time Operating Systems (FreeRTOS, Zephyr, ThreadX)
- Microcontroller families: ESP32/ESP-IDF, STM32 (HAL/LL), Nordic nRF5/nRF Connect SDK
- ARM Cortex-M architecture
- Communication protocols: UART, SPI, I2C, CAN, BLE, Wi-Fi

## Key Tools
- PlatformIO, ESP-IDF, STM32CubeIDE, nRF Connect SDK
- Debuggers: JTAG, SWD, OpenOCD, J-Link, ST-Link
- Logic analyzers and oscilloscopes
- Static analysis tools for embedded C/C++
- Memory leak detection and stack usage analysis
- RTOS tracing and profiling (SystemView, FreeRTOS stats)

## Integration
- Sensor drivers and peripheral interfaces
- Power management (sleep modes, low-power optimization)
- OTA update mechanisms and bootloaders
- Hardware abstraction layers (HAL/LL)
- Build systems and CI/CD for firmware
- Firmware size optimization and linker scripts
