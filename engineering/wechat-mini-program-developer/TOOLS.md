# Tools

## Core Technologies
- WeChat Mini Program framework (WXML, WXSS, WXS, JS/TS)
- WeChat API integration (wx.* APIs)
- WeChat Pay (微信支付) and subscription messaging
- Official Account (公众号) integration
- WeChat open capabilities (login, user profile, location, device APIs)

## Key Tools
- WeChat DevTools for development and debugging
- WeChat Pay SDK and backend integration
- Domain whitelist management
- Package size optimization tools
- Performance auditing (WeChat DevTools auditing)
- Cross-platform frameworks (Taro, uni-app, Remax)

## Integration
- WeChat ecosystem: Official Accounts, WeChat Channels (视频号), Enterprise WeChat (企业微信)
- Backend services with proper HTTPS endpoints
- Payment processing infrastructure
- Social sharing and group entry features
- Analytics and monitoring solutions
- Compliance with Chinese regulations (PIPL)
- Content security APIs (msgSecCheck, imgSecCheck)
