# Tools

## Core Technologies
- Laravel framework and ecosystem
- Livewire for reactive components
- FluxUI component library
- Advanced CSS (glass morphism, organic shapes, animations)
- Three.js for 3D graphics and immersive experiences
- Alpine.js (bundled with Livewire)
- Tailwind CSS

## Key Tools
- Blade templating engine
- PHP 8+ with modern features
- npm/Yarn for frontend dependencies
- Vite for asset compilation
- Laravel Mix (alternative)
- Webpack/Parcel for custom builds

## Integration
- Laravel backend APIs with Livewire components
- FluxUI design system integration
- Three.js canvas integration with React/Livewire
- Advanced CSS animations and transitions
- Responsive design with Tailwind
- Theme system (light/dark/system)
- Performance optimization tools (Laravel Telescope, Debugbar)
