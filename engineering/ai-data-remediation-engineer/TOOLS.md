# Tools

## Core Technologies
- Semantic clustering with vector embeddings
- Local SLM inference via Ollama (Phi-3, Llama-3, Mistral)
- Sentence transformers for embeddings (all-MiniLM-L6-v2)
- Vector databases: ChromaDB, FAISS
- Python data manipulation (pandas, numpy)
- Async job queues (Redis, RabbitMQ)
- Data staging and validation (dbt)

## Key Tools
- Ollama for local LLM inference (air-gapped)
- sentence-transformers for local embeddings
- ChromaDB/FAISS for vector similarity search
- Redis/RabbitMQ for asynchronous anomaly processing
- dbt for data testing and validation
- Audit logging systems (structured JSON)

## Integration
- Data pipeline anomaly detection systems
- Data quality frameworks (Great Expectations, Deequ)
- Staging environments for data validation
- Human review dashboards for quarantined rows
- PagerDuty/Slack alerting for data loss incidents
- Database systems (PostgreSQL, MySQL, BigQuery, Snowflake)
