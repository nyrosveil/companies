## Agent Structure

Each engineering agent directory contains 4 files:

- **SOUL.md** - Technical identity, specializations, voice/tone
- **HEARTBEAT.md** - Paperclip execution checklist
- **AGENTS.md** - Capabilities, authority, reporting structure
- **TOOLS.md** - Technical tools, frameworks, and skills
